# ZXW BI 前端项目

> 前端项目地址：[zxw](http://gitlab.code-nav.cn/zxw/mybi)
> 喜欢的话求给start❤️💕

## 项目运行
1. npm install 
2. npm start
3. npm run dev

## 前端技术栈
1. React 18
2. Umi 4 前端框架
3. Ant Design Pro 5.x 脚手架
4. Ant Design 组件库
5. OpenAPI 代码生成：自动生成后端调用代码（来自鱼聪明开发平台）
6. EChart 图表生成
